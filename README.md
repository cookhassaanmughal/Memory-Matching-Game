🧠 Memory Matching Game

A simple and interactive Memory Matching Game built using HTML, CSS, JavaScript, and jQuery. Flip the cards, match all pairs, and try to complete the game in the fewest moves possible.

🎮 How to Play

Click on a card to flip it.

Flip a second card to find its match.

If the symbols match, the cards stay revealed.

If they don’t match, they flip back.

Match all pairs to win the game.

✨ Features

4x4 game board (8 matching pairs)

Random shuffle every time the game starts

Move counter

Restart button

Visual feedback for flipped and matched cards

Clean and responsive design

🛠 Technologies Used

HTML5

CSS3

JavaScript

jQuery

🚀 How to Run the Project

Download or clone the repository.

Make sure all files (index.html, style.css, script.js) are in the same folder.

Open index.html in your browser.
